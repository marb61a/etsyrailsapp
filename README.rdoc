This is an Etsy Online Marketplace demo Rails App

It uses the following technologies

- Ruby 2.1.1
- Rails 4.1.1
- Bootstrap
- Paperclip\Imagemagick
- Dropbox for image hosting
- Figaro for protecting data
- Devise for user management
- Stripe for payment management
